DIYUSI主题Minecraft模组企划书

¶—禄存 2019年3月11日—

¶前言

从去年的 DIYUSI 热潮在 Cookie 圈内被引爆开始，不断有高质量创造涌现。惊异于 DIYUSI 的“交通警备员”人物设定形象在饼圈众人的一砖一瓦下迅速地丰满充实，变得“面白い[1]”的时候，却发现并没有什么相关的 Minecraft 模组。又因为正好接下来的半年里刚好有一些空闲时间，不想就这么浪费掉，我就开始试着搞点东西出来。

¶具体内容

¶模组构思

根据我洗澡的初步构思，整个模组的游玩路线与模因（meme）的创造与演化息息相关。由于 DIYUSI 是 MC 之外的模因，玩家需要先创造出 MC 原有的模因：Herobrine[2] 和 Notch[3]，扩大模因影响后就能在模因的边界上炸出一个洞，引入来自饼圈的模因。
玩家通过对机器（名字初设为“无限非概率发生器”，捏他《银河系搭车客指南》）塞入各种各样的人工合成物品提升概率。概率够高就能产生 Herobrine 和 Notch 的召唤物。通过恰当的召唤与杀死后会掉落关键合成物品（名字得和模因有关）。
关键物品可以用来击杀 SCP-910-JP，DIYUSI 使用交通标志攻击的原型之一。（可能需要先穿越到SCP特有维度？）之后用SCP-910-JP的掉落物和关键物品合成出各种武器和护甲。
玩家在穿有 DIYUSI 的服装并持有指挥棒时能够使用交通标志进行战斗。由于玩整合包的深刻体验，具体操作不会占用很多的快捷键，基本上鼠标加原版的几个行动键就 OK。有点像 AcademyCraft 那样，对着一个实体释放能力；也有技能点的熟练度一说。
可能会有 NEL，HNS，SIYUDI，CRN 之类的可扮演角色加入。

¶初步进度

自己也在学着写模组。目前弄出了一个可以用来撑交通标志的柱子，一个可以格挡近战（不免疫弹射物）的指挥棒，以及后者的合成配方。合成配方里面的“荧光粉”还可以当一个不错的照明光源。

¶程序开发

整个企划使用 ForgeModLoader，开发基于 Minecraft 1.12.2 版本的，围绕 DIYUSI 展开的游戏模组。编程语言自然是 Java 不过其它可用于 JVM[4] 上的语言，例如Scala都是可行的。
我自己对于模组材质、渲染和通信特别不了解，比较需要这方面的开发者。
暂时不需要比较厉害的美工建模等，不过 有能力的来者不拒。

¶进度规划

估计会在半年之后拿出一个没有完整流程，只有“能将就着看看”的关键物品的Alpha版本；在八月之前我都相对有空。
在那之后就开始完善整个模组，把完整的游戏流程加进来。一年应该能做完的吧？
然后就是修 bug，加更多有意思的东西。
真到了主线流程做完的那一天， 更高MC版本迁移也得提上日程。

¶人员安排

大家都是要恰饭的，一定要安排出多少时间不会强求，但是希望各位在没有意外情况的时候能尽量说到做到。我不太愿意去抓人催人，不是个合格的主催（笑）

¶投入与效果

大家尽量肝，我带头肝。不过还是学业生活为重。
这个模组在发布的时候可以成为“交通警备员”模组一类的，对过于饼干的要素规避一下应该是可以对外宣传出去的。要是事成了说不定口碑能变得很炄砒。关于开放捐助之类的可以到时候慢慢聊。这个更多的看大家的意愿和我自己的造化了。
如果在主流的模组圈推不开的话，最多圈内自嗨。

¶PS

使用交通标志战斗的角色真的很有意思呢。“规则的力量为我所用”之类的……

  2. Herobrine：一个在国内外 MC圈流行的，关于一个可怕的 Herobrine（简称 HIM）的Creepypasta。最初只是在一个主播用于炒作的视频。设定包括：神出鬼没、能使用红石机关等技术手段击杀玩家，在两格高一格宽的矿道里穿行，用神出鬼没的写有宣言的告示牌和书进行恐吓玩家。一般与 Notch 是死敌。 ↩

  3. Notch：原型是 MC开发者Markus Alexej Persson，设定包括：创世神，对玩家慷慨的神明，与 Herobrine 是死敌. ↩
